ABOUT:

This website serves primarily as an introduction to the world of vector spaces, one of the most fundamental concepts in linear algebra. We discuss some fundamental ideas surrounding vector spaces such as subspaces, linear dependence and independence, basis, dimension, the Rank-Nullity theorem, etc. in a semi-rigorous fashion. To begin, click on any of the links in the navigation bar above.

NOTE: The math used in this website incorporates the use of MathJax to make the implementation of the math much more appealing. Visit the website here if you have any concerns.

Made for MAT188, specifically for Chapters 3 and 5 in Nicholson's Linear Algebra with applications.
